# 🩺 Diabetes Prediction using Machine Learning

This project uses machine learning models to predict whether a patient is likely to have diabetes based on diagnostic health measures. The aim is to support early detection and assist in medical decision-making using data-driven insights.

---

## 📁 Project Structure


---

## 📊 Dataset

- **Name**: Pima Indians Diabetes Dataset
- **Source**: [Kaggle](https://www.kaggle.com/datasets/uciml/pima-indians-diabetes-database)
- **Features**:
  - Pregnancies
  - Glucose
  - BloodPressure
  - SkinThickness
  - Insulin
  - BMI
  - DiabetesPedigreeFunction
  - Age
- **Target**: `Outcome` (0 = No diabetes, 1 = Diabetes)

---

## 📌 Objectives

- Understand the dataset through EDA.
- Clean and preprocess the data.
- Train classification models to predict diabetes.
- Evaluate model performance.
- Build a simple prediction app (optional).
